# Finance Company Profile Monorepo

This project is a monorepo containing the backend and frontend for a Professional Financial Company Profile website.

## Project Structure

- **`/backend`**: Strapi CMS (Headless CMS) running on SQLite.
- **`/frontend`**: Next.js 14+ application with TypeScript, Tailwind CSS, and Lucide React.

## Getting Started

### Backend (Strapi)

1. Navigate to the backend folder:
   ```bash
   cd backend
   ```
2. Start the development server:
   ```bash
   npm run develop
   ```

### Frontend (Next.js)

1. Navigate to the frontend folder:
   ```bash
   cd frontend
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
